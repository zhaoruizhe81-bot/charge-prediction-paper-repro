# 训练运行命令

这份文档用于记录项目在 Windows 环境下从装环境到跑训练的完整命令，便于换电脑后直接复现。

## 1. 进入项目目录

```powershell
cd /d D:\自然语言处理\毕设\论文\res3_repro
```

如果使用 PowerShell：

```powershell
Set-Location D:\自然语言处理\毕设\论文\res3_repro
```

## 2. 创建与激活虚拟环境

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
```

## 3. 安装依赖

先升级 `pip`：

```powershell
python -m pip install --upgrade pip -i https://pypi.tuna.tsinghua.edu.cn/simple
```

安装 `CUDA 12.8` 版 PyTorch：

```powershell
pip uninstall -y torch torchvision torchaudio
pip cache purge
pip install --no-cache-dir --force-reinstall torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu128
```

安装项目其余依赖：

```powershell
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
```

如果遇到 `numpy/pandas` 兼容问题，再执行：

```powershell
pip install "numpy<2" --upgrade --force-reinstall -i https://pypi.tuna.tsinghua.edu.cn/simple
```

## 4. 验证 CUDA 环境

```powershell
python -c "import torch; print(torch.__version__); print(torch.version.cuda); print(torch.cuda.is_available()); print(torch.cuda.get_device_name(0) if torch.cuda.is_available() else 'no gpu')"
```

正常情况下应该看到：

- `torch.__version__` 类似 `2.11.0+cu128`
- `torch.version.cuda` 为 `12.8`
- `torch.cuda.is_available()` 为 `True`

## 5. 下载预训练 BERT

官方页面：

- `https://huggingface.co/hfl/chinese-bert-wwm-ext`

国内镜像：

- `https://hf-mirror.com/hfl/chinese-bert-wwm-ext`

安装下载工具：

```powershell
pip install -U huggingface_hub -i https://pypi.tuna.tsinghua.edu.cn/simple
```

下载到项目目录：

```powershell
huggingface-cli download hfl/chinese-bert-wwm-ext --local-dir .\chinese-bert-wwm-ext --resume-download --endpoint https://hf-mirror.com
```

也可以使用 Python 命令：

```powershell
python -c "from huggingface_hub import snapshot_download; snapshot_download(repo_id='hfl/chinese-bert-wwm-ext', local_dir='./chinese-bert-wwm-ext', endpoint='https://hf-mirror.com')"
```


## 7. 冒烟测试

适合 `RTX 2050 4GB` 这类小显存机器的最小测试命令：

```powershell
python .\scripts\train_deep_models.py `
  --data-dir .\data\processed_110_paper `
  --output-dir .\outputs_smoke\deep_models_fc `
  --models fc `
  --device cuda `
  --pretrained-model .\chinese-bert-wwm-ext `
  --epochs 1 `
  --train-batch-size 1 `
  --eval-batch-size 1 `
  --gradient-accumulation-steps 2 `
  --selection-metric accuracy `
  --max-train-samples 32 `
  --max-valid-samples 16 `
  --max-test-samples 16
```

如果显存不足，可以使用更保守的版本：

```powershell
python .\scripts\train_deep_models.py `
  --data-dir .\data\processed_110_paper `
  --output-dir .\outputs_smoke\deep_models_fc `
  --models fc `
  --device cuda `
```

## 8. 正式训练命令

```powershell

  --models fc `
  --device cuda `

  --epochs 4 `
  --train-batch-size 1 `
  --eval-batch-size 2 `

  --selection-metric accuracy
```

查看指标：


### 8.2 平层 BERT-RCNN


### 8.3 层次 BERT-FC

运行前需要先得到平层 `FC` 的最优权重。

```powershell

  --data-dir .\data\processed_110_paper `
```

## 11. 常见问题

### 11.1 `Torch not compiled with CUDA enabled`

说明装成了 CPU 版 `torch`，重新执行：

```powershell
pip uninstall -y torch torchvision torchaudio
pip cache purge
pip install --no-cache-dir --force-reinstall torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu128
```

### 11.2 找不到 `train_50k.jsonl`

说明还没有先执行数据预处理，需要先跑：

```powershell
python .\scripts\prepare_data.py `
  --data-dir ..\..\2018数据集 `
  --output-dir .\data\processed_110_paper `
  --target-size 50000 `
  --seed 42
```

### 11.3 显存不足

优先按下面顺序调小：

- 把 `--train-batch-size` 调成 `1`
- 把 `--eval-batch-size` 调成 `1`
- 把 `--gradient-accumulation-steps` 提高到 `4` 或 `8`
- 增加 `--max-length 192`
- 关闭浏览器、聊天软件、游戏等占显存程序
