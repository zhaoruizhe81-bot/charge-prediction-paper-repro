# res3 复现说明（代码+数据+结果）

本包用于完整复现 `res3` 最终结果，包含：
- 代码：`charge_prediction/`、`scripts/`
- 处理后数据：`data/processed_110/`
- 实验支撑指标：`outputs_cmp_110_seed66/`
- 最终表格：`outputs_cmp_110_res3opt/`

---

## 1) 环境安装

```bash
cd res3_repro
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

如遇你之前机器上的 NumPy/Pandas 二进制冲突，先执行：

```bash
pip install "numpy<2" --upgrade --force-reinstall
```

---

## 2) 一键复算最终表（推荐）

本包已自带关键中间指标，直接重建最终对照表：

最方便的方法是直接打开 notebook：

```bash
jupyter notebook 一键复现_res3.ipynb
```

然后直接点击 `Run All`。

如果你更习惯命令行，也可以执行：

```bash
python scripts/build_res3_table.py
```

输出目录：
- `outputs_cmp_110_res3opt/results_table.csv`
- `outputs_cmp_110_res3opt/results_table_contrast.csv`
- `outputs_cmp_110_res3opt/requirements_check.md`

---

## 3) 重新训练（可选，耗时）

### 3.1 机器学习三模型 + 层次融合

```bash
python scripts/train_ml_baselines.py \
  --data-dir data/processed_110 \
  --output-dir outputs_cmp_110_seed66/ml_baselines \
  --models svm sgd pa \
  --seed 66
```

### 3.2 深度学习层次分类（示例）

```bash
python scripts/train_deep_hierarchical.py \
  --data-dir data/processed_110 \
  --output-dir outputs_cmp_110_seed66/deep_hierarchical_fc \
  --fine-model-type fc \
  --coarse-model-type fc \
  --seed 66
```

> 深度训练需GPU，耗时较长；论文复现实操建议优先用“第2步复算最终表”。

## 5) 当前仓库口径补充

本复现包内置的 `data/processed_110/` 和 `outputs_cmp_110_seed66/` 仍对应当前 `res3` 结果口径，可直接复算最终表。

如果你需要重新构建论文口径的 `3` 粗类数据，请使用 `scripts/prepare_data.py` 生成 `data/processed_110_paper/`。该脚本现在默认使用更稳的 `rebuild_stable` 策略：先筛标签，再从全量样本池重建分层切分，尽量避免验证集和测试集缺失标签；当原始支持度不足时，最终保留标签数可能少于 `110`。
